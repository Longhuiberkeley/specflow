# SpecFlow Status

Show the current project state.

## Usage

- `/cmd` — Displays current phase, artifact counts by type/status, broken links, orphan artifacts, missing verification pairs, and unresolved suspect flags.

This is the first command to run after any skill workflow to understand where the project stands.
