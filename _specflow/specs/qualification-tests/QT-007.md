# Cross-Platform and Compliance Qualification

## Test Reference

`tests/test_cross_platform_compliance.py`

## Test Cases

### TC-001: Baseline Creation and Comparison
- **Input**: Create baseline, modify artifacts, create second baseline
- **Expected**: Diff between baselines shows exact changes
- **Verification**: All changes detected, no false positives

### TC-002: Standards Pack Architecture
- **Input**: Install standards pack, run gap analysis
- **Expected**: Pack loaded, compliance gaps identified against installed standard
- **Verification**: Gap analysis results match expected non-compliant areas

### TC-003: Retroactive Change Records
- **Input**: Git history with commits, run document-changes
- **Expected**: Change records generated from git history and impact-log
- **Verification**: Records accurately reflect commit history

### TC-004: RBAC via Git Hooks
- **Input**: Configure roles, attempt operations outside role
- **Expected**: Unauthorized operations blocked by hook
- **Verification**: CODEOWNERS generated correctly, hooks enforce permissions

### TC-005: ReqIF Import and Export
- **Input**: Export artifacts to ReqIF, re-import
- **Expected**: Round-trip preserves all artifact data
- **Verification**: Imported artifacts match original source

## Acceptance Criteria

1. All test cases pass deterministically
2. Tests run with no network or LLM calls
3. Tests use `tmp_path` fixtures for isolation
