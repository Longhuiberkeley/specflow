# Artifact Library Unit Tests

## Test Reference

`tests/test_artifacts.py`

## Scope

Tests for `lib/artifacts.py`:
- `compute_fingerprint()` — SHA256 of normative content
- `parse_artifact()` — YAML frontmatter + Markdown body extraction
- `find_orphans()` — Links with non-existent targets
- `find_missing_v_pairs()` — Missing V-model verification links
- `trace_chain()` — Upstream/downstream link traversal
- `compute_chain_depth()` — Longest path from standard to test

## Acceptance Criteria

1. All tests pass deterministically
2. No network or LLM calls
3. Uses `tmp_path` for filesystem isolation
