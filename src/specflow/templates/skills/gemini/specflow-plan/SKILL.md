# SpecFlow Plan

Break down approved requirements into architecture, detailed design, and user stories.

## Workflow

1. **Phase Gate Check** — Confirm REQ artifacts have `status: approved`.
2. **Architecture Proposal** — Create ARCH artifacts.
3. **Detailed Design** — Create DDD artifacts.
4. **Story Breakdown** — Create STORY artifacts linking to specs.
